UCG Web Dev Requirements

  • Website must be deployed on github and accessible through other computers
  • All files must produce no errors when analysed on Code Pen
  • HTML must have each of the following:
    • Exactly 1 h1 tag
    • At least 3 h2 tags
    • Each h2 tag must have a p tag with sufficient text about the h2 tag
    • Either an ordered list or unordered list with at least 3 items
    • At least one of the following: an image, audio, video, or another website
  • CSS must have each of the following:
    • A tag selector and either a class or id selector
    • Change the color of some text
    • Change the color of one of the backgrounds, or use a background image
    • Change the alignment of at least one piece of text
    • Change the font size of at least one piece of text
